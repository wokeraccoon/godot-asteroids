# godot-asteroids
Asteroids clone made with Godot 3.3. Assets by kenney

# How to import

1. Download as zip or clone it somewhere
2. Open godot and click on import
3. Direct it to the .zip or folder
4. Done

# What works so far

-Player Movement
-Asteroid spawning
